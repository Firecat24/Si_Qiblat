# SiKiblat - Smart Qibla Finder 🧭

**SiKiblat** adalah instrumen penentu arah kiblat berbasis mikrokontroler yang dirancang untuk memberikan akurasi tinggi dalam menentukan arah secara praktis. Alat ini menggabungkan sensor magnetometer dengan algoritma native untuk penggunaan lapangan.

## ✨ Fitur Utama
* **Real-Time Direction:** Menentukan arah kiblat secara instan.
* **Algoritma Native:** Perhitungan matematis bola bumi tanpa perlu koneksi internet.
* **Portable Design:** Ringkas dan mudah dibawa untuk kebutuhan masyarakat.

## 🛠️ Teknologi & Hardware
* **Microcontroller:** Arduino Series.
* **Sensors:** Digital Compass (Magnetometer).
* **Programming:** C++ (Arduino Sketch).
